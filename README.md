# Multi-Agentic Content Generation System

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

An automated content generation system that creates structured product content including FAQs, product descriptions, and comparison pages.

## Features

- Generates 31+ categorized questions
- Creates structured JSON outputs
- Modular, agent-based architecture
- Support for multiple content types

## Getting Started

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Run the content generation
npm start
